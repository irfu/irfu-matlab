Non-Standard Operations (NSO) List

Manually written Non-Standard Operations (NSO) List used by BICAS (SolO/RPW/BIAS calibration and data processing). Contains "events", i.e. time intervals, each one labelled with a standardized string (NSO ID, or RCS NSO ID) that BICAS can interpret. BICAS uses this list to modify how datasets are processed, e.g. by setting zVariables QUALITY_FLAG and/or L2_QUALITY_BITMASK, replacing data with fill values, and/or omitting data from L3.
